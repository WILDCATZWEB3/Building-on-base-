What is Base?

Base is a Layer 2 blockchain built on Ethereum by Coinbase, offering faster and cheaper transactions. It's designed to bring the next billion users on-chain with seamless user experience and full Ethereum compatibility.

Who Created Base?

Base was created by Coinbase, led by Jesse Pollak. It uses the open-source OP Stack developed by Optimism.

When Was Base Launched?

Base was announced in February 2023, testnet launched the same month, and its public mainnet went live on August 9, 2023.

Why Base?

  • Cheap and fast Ethereum-powered transactions
  • Built-in access to 100M+ Coinbase users
  • Developer-friendly and open-source
  • No native token — uses ETH

How Base Works

Base uses Optimistic Rollups via the OP Stack. Transactions are processed off-chain, then posted to Ethereum for security. It inherits Ethereum’s security while scaling throughput and reducing gas fees.

Cool Facts

  • Backed by Coinbase but aims to be decentralized
  • Part of Optimism's Superchain vision
  • Open to all builders — no token needed
